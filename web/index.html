  • Q2: Configuration for GnuPG

    For GnuPG 2, you can put following line in ~/.gnupg/dirmngr.conf: 

    keyserver hkps://pgp.ustc.edu.cn
    Then run gpgconf --reload dirmngr to reload dirmngr.
